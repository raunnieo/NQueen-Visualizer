N-Queens Problem

The N-Queens problem is a classic chess puzzle where N queens must be placed on an NxN chessboard so that no two queens threaten each other.

Backtracking

This solver uses a backtracking algorithm. Backtracking is an algorithmic technique that considers searching every possible combination in order to solve a computational problem.
